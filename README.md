# LinerSync Chunk 1

This is the first rebuild chunk for the real React/Vite app base.

Included:
- React + Vite + TypeScript structure
- S25 Ultra mobile-first layout
- One Help button
- Home / New Project / Current Project flow
- Local project save in browser storage

Not included yet:
- Repair / Roll / Panel / Seam capture
- Supabase cloud backup wiring
- Mythos command history
- Export layer

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
